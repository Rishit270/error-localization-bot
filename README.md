# 🛠 Error Localization Bot

A Python Tkinter GUI tool that analyzes runtime errors, detects suspect code lines, and suggests fixes automatically.

##  Features

- Detect Python runtime errors
- Stack trace analysis
- Finds suspect line
- Suggests fixes
- Source code viewer
- Save reports

##  Tech Stack

- Python
- Tkinter
- Regex
- Subprocess

##  Run Project

```bash
python main.py
```
